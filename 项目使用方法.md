## Git 拉取与合并更新

```Markdown
- **origin（您的仓库）**: https://github.com/pelioo/CowAgent.git
- **upstream（上游主仓库）**: https://github.com/zhayujie/chatgpt-on-wechat.git

您想要从 **upstream**（上游主仓库）拉取更新合并到本地。

---

## 合并上游仓库计划

### 执行步骤

1. **获取上游仓库的最新代码**
   ```bash
   git fetch upstream
   ```

2. **查看上游有哪些新提交**
   ```bash
   git log upstream/master --oneline -10
   ```

3. **将 upstream/master 合并到本地 master**
   ```bash
   git merge upstream/master
   ```

4. **如果有冲突**，Git 会提示，您需要手动解决冲突后 `git add` 提交

---

**注意**：当前您的本地 `master` 已经是最新的（与 `origin/master` 同步）。如果您之前没有设置 `upstream` 远程，可能需要先确认 upstream 是否已配置好。

请确认是否要执行以上步骤？切换到 **Act 模式** 后我可以帮您执行合并操作。
```
