## 1. 项目简介

本项目是一个基于大语言模型的智能对话 Agent 后端服务器，支持多种前端渠道（channel）接入，包括：
- 微信、企业微信
- 飞书、钉钉
- Web 网页
- 终端
- 等其他渠道

## 3. 安装部署

### 3.1 克隆项目

```bash
# 克隆自己的仓库（origin）
git clone https://github.com/pelioo/CowAgent.git
cd CowAgent

# 添加上游仓库（可选，用于获取更新）
git remote add upstream https://github.com/zhayujie/chatgpt-on-wechat.git
```

### 3.2 创建虚拟环境

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/macOS:**
```bash
python -m venv venv
source venv/bin/activate
```

### 3.3 安装依赖

```bash
pip install -r requirements.txt
```

### 3.4 配置环境

1. 复制配置文件：
```bash
cp config-template.json config.json
```

2. 编辑 `config.json`，配置以下必要项：
   - `model`: 使用的模型类型
   - `api_key`: API 密钥
   - 渠道相关配置

## 5. Git 操作指南

### 5.1 仓库说明

| 仓库 | 地址 | 说明 |
|------|------|------|
| origin | https://github.com/pelioo/CowAgent.git | 您的个人仓库 |
| upstream | https://github.com/zhayujie/chatgpt-on-wechat.git | 上游主仓库 |

### 5.2 拉取上游更新

```bash
# 1. 获取上游仓库最新代码
git fetch upstream

# 2. 查看上游更新内容（可选）
git log upstream/master --oneline -10

# 3. 切换到本地主分支
git checkout master

# 4. 合并上游更新
git merge upstream/master

# 5. 如有冲突，手动解决后提交
git add .
git commit -m "Merge upstream changes"
```

### 5.3 推送到个人仓库

```bash
# 推送到 origin
git push origin master
```

> **注意**: 首次操作前请确保已配置好 Git 用户名和邮箱。

## 6. 常见问题

### Q1: 启动时报错 "ModuleNotFoundError"

**解决方法**: 确保已激活虚拟环境并安装依赖
```bash
# 激活虚拟环境
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/macOS

# 重新安装依赖
pip install -r requirements.txt
```

### Q2: 配置文件在哪里？

**答**: 项目根目录下的 `config.json`（需从 `config-template.json` 复制创建）

### Q3: 如何查看日志？

**答**: 
- 控制台直接输出
- 查看 `logs/` 目录下的日志文件

### Q4: 如何添加新的渠道？

**答**: 参考 `channel/` 目录下的现有实现，继承 `Channel` 基类实现即可。

### Q5: 如何联系项目维护者？

**答**: 
- GitHub Issues: https://github.com/pelioo/CowAgent/issues
- 上游项目: https://github.com/zhayujie/chatgpt-on-wechat
